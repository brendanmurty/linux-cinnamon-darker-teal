# Darker Teal theme for Cinnamon

Custom Cinnamon theme for Linux Mint based on the default `Mint-Y-Dark-Teal` theme.

Note that this theme:

- Assumes the Panel is filling the whole width of the screen
- Assumes the Panel is aligned to the bottom of the screen
- Assumes the default 'Grouped Window List' Applet is used in the Panel

## Initial Setup

Review the contents of [bin/setup.sh](bin/setup.sh) to see how this customisation works.

Then if you want to apply it to your system, run the initial setup script:

```bash
./bin/setup.sh
```

## Apply Changes

After making changes to the theme files, you can apply them to the system with:

```bash
./bin/apply.sh
```
